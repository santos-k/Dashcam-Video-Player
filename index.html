Features

Everything for Dashcam Review

From quick review to professional export — all in one lightweight desktop app

📷

Dual Camera Sync

Auto-detects and pairs front & back camera files by timestamp with ±5s tolerance. Supports video_front/video_back folder structure and F/B filename suffixes.

🎬

3 Layout Modes

Side-by-side, Stacked, and Picture-in-Picture. PIP is fully draggable and resizable with position memory. Switch instantly with keys 1/2/3.

⏱

Variable Speed (0.1x–5x)

11 speed levels from 0.1x slow-motion to 5x fast-forward. Speed persists across clip changes. Perfect for reviewing timelapse dashcams.

🌎

GPS & Interactive Map

OpenStreetMap sidebar with device location, coordinate search, 3 tile layers (Standard, Topo, HOT), and links to Google Maps & OSM.

💾

FFmpeg Export

Export composed videos in any layout mode with H.264 encoding. Sync offset applied automatically. Real-time progress tracking.

📋

Batch Save with Progress

Select multiple clips and save with a live counter showing "2/6 saved". Supports locked clip detection and per-camera filtering.

🎹

Independent Audio Control

Mute front and back cameras independently. Fine-tune sync offset with a ±5000ms slider for perfect audio-video alignment.

⌨

20+ Keyboard Shortcuts

Full keyboard-first workflow: playback, seek, speed, layout, mute, panels, export, save, sort, and fullscreen. All toggles work in loop.

🛠

Smart Auto-Hide UI

Controls disappear after 5 seconds of inactivity during playback, letting the video fill the entire screen. Move the mouse to bring them back.
