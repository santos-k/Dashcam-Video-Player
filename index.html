Features

Everything for Dashcam Review

From quick review to professional export — all in one lightweight desktop app

📷

Dual Camera Sync

Auto-detects and pairs front & back camera files by timestamp with ±5s tolerance. Supports video_front/video_back folder structure and F/B filename suffixes.

🎬

3 Layout Modes

Side-by-side, Stacked, and Picture-in-Picture. PIP is fully draggable and resizable with position memory. Switch instantly with keys 1/2/3.

⏱

Variable Speed (0.1x–5x)

11 speed levels from 0.1x slow-motion to 5x fast-forward. Speed persists across clip changes. Perfect for reviewing timelapse dashcams at real-time pace.

🌎

GPS & Interactive Map

OpenStreetMap sidebar with device location, manual coordinate search, multiple tile layers (Standard, Topo, HOT), and links to Google Maps & OSM.

💾

FFmpeg Export

Export composed videos in any layout mode with H.264 encoding. Applies sync offset automatically. Progress bar with percentage tracking.

📋

Batch Save with Progress

Select multiple clips to save with a real-time counter showing "2/6 saved". Supports locked (protected) clip detection and per-camera filtering.

🎹

Independent Audio Control

Mute front and back cameras independently. Front audio is used for export by default. Fine-tune sync offset with ±5000ms slider.

⌨

20+ Keyboard Shortcuts

Full keyboard control: play/pause, seek, speed, layout, mute, panels, export, save, sort, fullscreen. All toggle shortcuts work as expected.

🛠

Smart UI

Auto-hiding controls after 5s of inactivity, video fills the full screen. Sortable clip drawer, locked clip badges, and responsive layout.
