Features

Everything for Video Review

From dashcam footage to any video — powerful playback in a lightweight desktop app

New in v3.0
📡

Wi-Fi Dashcam (Full API)

Connect to Onelap dashcam over Wi-Fi. Full REST API: browse files, delete, download, settings, snapshot, recording control. Auto-loads clips into main player.

New in v3.0
🔍

Zoom In/Out (up to 8x)

Pinch/scroll zoom on each video pane independently. Keyboard shortcuts = - 0. Zoom indicator badge. Reset with one click.

New in v3.0
🔊

Volume Sliders

Per-camera volume control with vertical slider popup on hover. Click to mute/unmute, drag for fine control. Separate F/B sliders for paired clips.

New in v3.0
📂

Smart Clip Grouping

Group clips by date, type (Paired/Video/F-only/B-only), or source (Local/WiFi). Collapsible groups with select-all per group. Works in both list and grid view.

New in v3.0
🎬

Universal Video Player

Play 19+ formats: MP4, MKV, AVI, TS, WebM, FLV, WMV, M4V, 3GP, MTS, VOB, and more. Recursive folder scanning picks up all videos.

New in v3.0
📷

5 Layout Modes

Side-by-side, Stacked, PIP, Front Only, Back Only. Switch with 1–5 keys. Front/Back Only for focused single-camera review.

Core
📷

Dual Camera Sync

Auto-pairs front & back camera files by timestamp with ±5s tolerance. Supports dashcam folders, F/B suffixes, and nested directories.

Core
🎨

Thumbnail Clip Browser

Browse clips with thumbnails in grid or list view. Multi-select, batch delete, clip durations, WiFi badge, 6 sort modes, and collapsible group headers.

Core
⌨

35+ Customizable Shortcuts

Every action has a keyboard shortcut, all remappable via built-in editor. Zoom, volume, layout, WiFi panel — everything at your fingertips.

🌎

GPS & Interactive Map

OpenStreetMap sidebar with device location, coordinate extraction, 3 tile layers, and links to Google Maps & OSM.

💾

FFmpeg Export

Export composed videos in any layout with H.264 encoding. Supports all 5 layouts including Front Only and Back Only. HTTP URL export for WiFi clips.

🔔

Modern Notifications

Top-right slide-in notifications with auto-dismiss. Green for success, red for errors, amber for warnings. Clean pill design with close button.

Changelog

What's New in v3.0

17 commits of new features, bug fixes, and improvements

  • New Wi-Fi dashcam: full Onelap REST API with 16 endpoints (files, settings, delete, snapshot, recording)
  • New Universal video player: 19+ formats (MP4, MKV, AVI, TS, WebM, FLV, WMV, M4V, 3GP, MTS, VOB...)
  • New Recursive folder scanning: finds videos in all subdirectories automatically
  • New Zoom in/out up to 8x with pinch/scroll, keyboard shortcuts (= - 0), and reset button
  • New Per-camera volume sliders with vertical popup on hover, independent F/B control
  • New 5 layout modes: Side-by-side, Stacked, PIP, Front Only, Back Only (1–5)
  • New Smart clip grouping: by date, type, or source with collapsible sections and group select-all
  • New 6 sort modes: Date, Name, Duration — ascending and descending
  • New Clip durations probed via ffprobe and displayed in both list and grid views
  • New WiFi clips auto-load into main clip list with front/back pairing
  • New Side-by-side RTSP live view with front/back camera labels
  • New N shortcut to toggle Wi-Fi dashcam panel
  • New Modern pill-style notifications (green/red/amber) replacing snackbars
  • Improved Controls toggle on click (no auto-hide timer), always visible by default
  • Improved WiFi file delete via /app/deletefile API (was faking success before)
  • Fixed Delete removing wrong clips when sorted newest-first
  • Fixed Controls not reappearing after video loads
